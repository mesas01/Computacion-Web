Desarrollo del proyecto

Esta sección presenta los endpoints principales de nuestra API REST, diseñados para la gestión de usuarios del sistema. Cada endpoint implementa las mejores prácticas de desarrollo web y garantiza una comunicación eficiente entre el cliente y el servidor.

GET
/api/usuarios

Obtener Lista de Usuarios

Este endpoint permite obtener la lista completa de usuarios registrados en el sistema. Incluye funcionalidades de paginación y filtrado para optimizar el rendimiento.

Parámetros de Consulta

page (integer, opcional)
Número de página para la paginación. Por defecto: 1
limit (integer, opcional)
Cantidad de registros por página. Por defecto: 10

Respuesta

200 Success
{ "data": [ { "id": 1, "nombre": "Juan Pérez", "email": "juan@email.com", "fecha_registro": "2024-01-15" } ], "pagination": { "page": 1, "limit": 10, "total": 25 } }
POST
/api/usuarios

Crear Nuevo Usuario

Permite registrar un nuevo usuario en el sistema. Incluye validaciones de integridad y verificación de duplicados para garantizar la consistencia de los datos.

Cuerpo de la Solicitud

nombre (string, requerido)
Nombre completo del usuario (mínimo 2 caracteres)
email (string, requerido)
Dirección de correo electrónico válida y única
password (string, requerido)
Contraseña (mínimo 8 caracteres)

Respuesta

201 Created
{ "message": "Usuario creado exitosamente", "data": { "id": 26, "nombre": "María García", "email": "maria@email.com", "fecha_registro": "2024-08-03" } }
