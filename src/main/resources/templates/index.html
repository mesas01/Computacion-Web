Proyecto

Arrienda tu finca es una aplicación web que conecta a personas interesadas en arrendar fincas con propietarios que desean ofrecerlas. El sistema está diseñado para dos tipos de usuarios principales: el Usuario Arrendatario (quien ofrece la finca) y el Usuario Arrendador (quien la busca y arrienda).

El Usuario Arrendatario puede gestionar propiedades (crear, editar, eliminar, actualizar), revisar solicitudes de arriendo, aceptarlas o rechazarlas, verificar pagos realizados y calificar al arrendador luego del proceso.

El Usuario Arrendador puede buscar fincas por ubicación, solicitar arrendamientos, realizar pagos una vez aprobada la solicitud y calificar tanto la finca como al arrendatario después del uso.

La plataforma requerirá un sistema sólido de autenticación y autorización, interfaces intuitivas y una arquitectura que permita una experiencia fluida y segura. El enfoque se centra en facilitar el proceso de arrendamiento para ambas partes, garantizando eficiencia, transparencia y satisfacción.

Objetivos

Facilitar el arriendo de fincas mediante una plataforma web intuitiva y accesible.

Gestionar propiedades fácilmente, permitiendo a los arrendatarios crear, editar y controlar sus publicaciones.

Optimizar la experiencia del usuario arrendador, con filtros de búsqueda, solicitud de arriendos y pagos en línea.

Garantizar seguridad y confianza mediante autenticación, control de pagos y sistema de calificaciones.
